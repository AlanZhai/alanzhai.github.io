Hexo博客添加Algolia搜索

发表于2021-09-17|更新于2022-01-07
|阅读量:

Algolia介绍：

​ Algolia原理是将数据库的信息以JSON的格式上传到Algolia服务器，然后通过调用Algolia的API进行搜索，可以理解为云搜索，可以实现实时搜索

  1. 注册并登陆Algolia官网

  2. 我们新建一个index，名字随意取

  3. 在侧边栏依次选择API Keys–All Api Keys，然后就需要新建一个API，我们点击New API Key

  4. 我们选择刚刚创建的index，然后添加权限search，addObject和deleteObject三个权限

  5. 复制保存好创建的API Key

  6. 安装Algolia搜索功能

    • 在Hexo根目录下安装hexo-algolia插件

    • 在Hexo根目录的_config.yml中加入algolia的配置

    • 设置环境变量

    • 上传数据到algolia

    • 上传成功

  7. 在主题配置文件下开启algolia

    • 这里的配置仅适用于butterfly主题，其他主题可以参照主题官方的配置文档
